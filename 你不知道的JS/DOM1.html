2.Node 关系

所有节点都存在关系，并且有个childrenNodes属性 访问所有内部节点

每个节点都一个parentNode属性指向文档书中的父节点

此外他们具有previousSibling和nextSibiling属性访问父节点下的同级属性

节点有hasChildNode方法判断是否有子节点

            p.parentNode // body
            p.previousSibling // 空字符
            p.nextSibling // 空字符
